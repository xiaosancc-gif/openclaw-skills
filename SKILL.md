# openclaw-skills-install

一键安装阿飞哥 OpenClaw Skills 完整包。

## Trigger Phrases

- "安装阿飞哥的Skills包"
- "安装我的Skills包"
- "同步Skills"
- "安装全部Skills"
- "openclaw-skills"

## What This Skill Does

自动从 GitHub 仓库克隆 Skills 安装包并执行安装：
- 91 个 Skills 完整清单
- 12 个 GitHub 项目一键安装命令
- 插件（mem9、飞书）安装

## 安装命令

### Skills（clawhub）
```bash
npx clawhub install <skill-name>
```

### GitHub 项目
```bash
npx agent-reach install <repo>
```

### 插件
```bash
openclaw plugins install @larksuite/openclaw-lark
openclaw plugins install @mem9/mem9
```
